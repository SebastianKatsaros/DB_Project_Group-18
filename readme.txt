Η εφαρμογή αυτή αφορά τη διαχείριση ενός ζωολογικού κήπου. Εκτελεί τυπικές λειτουργίες που αφορούν τον κήπο. Το αρχείο που κατεβάσατε από το github περιλαμβάνει έναν κώδικα σε γλώσσα προγραμματισμού Python και τη βάση δεδομένων zoo_database.db. Δεν απαιτείται η εγκατάσταση κάποιας επιπλέον βιβλιοθήκης για να εκτελεστεί ο κώδικας της εφαρμογής. Τρέξτε τον κώδικα με όποια εφαρμογή εκτέλεσης κώδικα Python έχετε ήδη εγκαταστήσει.
